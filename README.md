## 📖 Descripción

Página web personal en HTML/CSS estático, deployada en GitHub Pages. Sirve como punto de entrada único a mi presencia online: enlaza con mis proyectos en GitHub, mis apps deployadas en Streamlit Cloud y mis perfiles profesionales (LinkedIn, email).

**¿Por qué un portfolio web aparte si ya tengo perfil de GitHub?**

- ✅ URL **personalizada** y memorable: `tu-usuario.github.io`
- ✅ Diseño **propio**: cuento mi historia a mi manera
- ✅ Centraliza enlaces a apps, repos, LinkedIn → todo en un sitio
- ✅ Una sola URL en CV, firma de email, LinkedIn, etc.

## ⚙️ Cómo personalizarlo

1. Abre `index.html` y reemplaza:
   - `Tu Nombre` por tu nombre
   - `tu-usuario` por tu usuario de GitHub (en TODAS las URLs)
   - `tu@email.com` por tu email
   - El texto de "Sobre mí"
   - Las **cards de proyectos** con tus proyectos reales (URLs de GitHub y Streamlit)

2. Abre `styles.css` y, si quieres, cambia los colores en la sección `:root`:
   ```css
   --color-primary: #6366f1;   /* tu color principal */
   --color-accent: #06b6d4;    /* tu color de acento */
   ```

3. Commit + push y GitHub Pages se actualiza solo en 1-2 minutos.

---

## 🚀 Cómo se deployó

1. Crear repo en GitHub con el nombre **EXACTO** `tu-usuario.github.io`
2. Subir `index.html` y `styles.css` a la raíz
3. **Settings → Pages → Source: Deploy from branch → main**
4. ¡Listo! La URL `https://tu-usuario.github.io` funciona automáticamente
